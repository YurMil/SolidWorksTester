## 1. Purpose

Batch creation of SOLIDWORKS drawing documents (`.SLDDRW`) from part files (`.SLDPRT`) using a user-selected drawing template (`.DRWDOT` or `.SLDDRW`).

- Output drawing is saved next to the source part.
- Part geometry type is detected automatically.
- The source model is read-only; annotations are added on the drawing only.

---

## 2. Supported drawing pipelines

The application selects one pipeline per part based on automatic model analysis.

| ID | Part type | Views | Primary annotations |
| --- | --- | --- | --- |
| P-01 | Flat plate (sheet or EST PLATE) | Front, top, right, isometric | Sub-kinds: overall/holes, round OD, rounded-end, **arc-sector (R_in/R_out/angle/strip)**, flange, baffle |
| P-02 | Bent sheet metal | Front, top, right, flat pattern | Bend dims, holes, positions, cutouts, thickness, overall sizes |
| P-03 | Cylindrical (pipe, tube, cylinder) | Front, top, right, isometric | Centerlines, OD, wall thickness, ID (parentheses), length, model dims |

### P-01 — Flat plate

- Three orthographic views plus isometric view (no flat pattern when there are no bends).
- **Generic:** overall sizes, optional symmetry centerlines, thickness, holes, cutouts, fillets/chamfers.
- **Round disc:** outer diameter, centerlines, thickness on side views.
- **Rounded-end / segment:** overall, outer arc, tip, holes.
- **Arc-sector:** inner/outer radii, sector angle, radial strip width, bbox, hole Ø + two coordinates (e.g. EST-P61728).
- **Flange / baffle:** dedicated nested pipelines.
- Duplicate dimensions removed across views; sheet layout normalized when views exceed the sheet.

### P-02 — Bent sheet metal

- Three orthographic views plus flat pattern view.
- Bend radii/angles and hole-related dimensions where geometry is detected.
- Duplicate dimensions removed across views.
- Sheet scale adjusted when required.

### P-03 — Cylindrical parts

- Three orthographic views plus isometric view.
- Drawing centerlines and center marks (SOLIDWORKS drawing commands; model is not modified).
- End-face view: outer diameter, wall thickness, inner diameter shown in parentheses.
- Side views: overall length.
- Model dimensions marked for drawing imported when available.
- Chamfer annotations when detected.
- Duplicate and false hole-style dimensions cleaned up.

---

## 4. Known limitations

- Not a substitute for engineering review or controlled document release.
- Assemblies, weldments, and unclassified geometry may not be supported.
- Automatic view layout and dimension placement may require manual adjustment.
- Drawing quality depends on template, model preparation, and SOLIDWORKS settings.

---

## 5. Manual verification

All generated drawings **must be reviewed manually** before release. Automated output does not guarantee compliance with internal design documentation rules or final drawing quality.
